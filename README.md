# CloudSilos

CloudSilos is a cloud-based file storage backend built using FastAPI, SQLite, Redis, and Google Cloud Storage. The system uses SHA-256 hashing-based deduplication to prevent storing duplicate files and optimize cloud storage usage.

## Tech Stack
- Python
- FastAPI
- SQLite
- Google Cloud Storage

## Features
- JWT Authentication
- File Upload & Download APIs
- SHA-256 File Deduplication
- Google Cloud Storage Integration
